# Acceptance Criteria

## User Story 1
As a lecturer, I want to record attendance so that I can track student participation.

### Acceptance Criteria
- Lecturer can enter student name
- Lecturer can mark present or absent
- Attendance is stored correctly
- Confirmation message is displayed

---

## User Story 2
As a lecturer, I want to search for students so that I can quickly find records.

### Acceptance Criteria
- Lecturer can search using student name
- System displays matching student
- Invalid searches show error message

---

## User Story 3
As a lecturer, I want attendance reports so that I can monitor attendance trends.

### Acceptance Criteria
- System generates attendance summary
- Report displays percentage attendance
- Lecturer can view saved records
